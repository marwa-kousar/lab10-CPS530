Using Tailwind CSS Framework

How Tailwind CSS Was Installed

Tailwind CSS was installed using the CDN link for simplicity. By including the following line in the <head> section of the HTML file: 

                <link href="https://cdn.jsdelivr.net/npm/tailwindcss@2.2.19/dist/tailwind.min.css" rel="stylesheet">

How the Page Was Built

The page is structured with a header, main content, and footer. Tailwind CSS utility classes were used to:

  • Style the header with a blue background and white text.
  • Add padding and margin for a clean layout.
  • Use responsive classes to ensure the design looks good on all devices.

Difficulties Encountered

One difficulty was ensuring the correct use of Tailwind classes without conflicts. Additionally, hosting on GitHub Pages required careful attention to ensure the relative paths for CSS files were correct.

These were resolved by carefully testing and referring to the Tailwind CSS documentation.
